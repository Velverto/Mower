Installation guide:
This is a console application using Java.
A)Requirement:
	Java SE Development Kit: http://www.oracle.com/technetwork/java/javase/downloads/index.html
  Apache Ant: https://ant.apache.org/
	
B)Install
1: Download or clone the files to a directory
2: Open the shell, go to the directory with files downloaded. Make sure you end up in same directory with build.xml
3: Run the buildscript by typing
		ant compile jar run
